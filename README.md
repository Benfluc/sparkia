# SparkIA

Site institucional da SparkIA, empresa de consultoria e tecnologia focada em dados e inteligência artificial para empresas. O site apresenta os serviços oferecidos, portfólio e informações de contato.

Acesse o site publicado: https://benfluc.github.io/sparkia/

## Sobre

A SparkIA ajuda empresas a extrair valor de seus dados e a aplicar inteligência artificial em processos de negócio, com foco em soluções práticas e de baixo atrito para adoção.

## Serviços

- Insights valiosos com dados: extração de informações relevantes para conhecer melhor o negócio e o público
- Chatbots para atendimento: automatização do atendimento ao cliente
- Incremento de vendas e serviços: sistema de recomendação de produtos e análise preditiva de vendas
- Automação de processos: geração automática de textos, posts e anúncios, resposta automática de e-mails, extração de informações de PDFs, entre outros

## Tecnologias

- Jekyll (gerador de site estático)
- Tema Creative, da Start Bootstrap
- Bootstrap, HTML, CSS e JavaScript
- Publicado via GitHub Pages

## Requisitos

- Ruby
- Bundler
- Jekyll

## Como executar localmente

Clone o repositório:

```bash
git clone https://github.com/Benfluc/sparkia.git
cd sparkia
```

Instale as dependências:

```bash
bundle install
```

Inicie o servidor local:

```bash
bundle exec jekyll serve
```

O site ficará disponível em http://localhost:4000.

## Deploy

O site é publicado automaticamente via GitHub Pages a partir da branch master.
